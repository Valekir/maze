## Описание проекта
Игра "Лабиринт", написанная с используем клиент-серверной архитектуры. Используется собственный протокол, разработанный поверх TCP с использованием стандартных C++ сокетов. Сервер поддерживает неограниченное
(в пределах допустимых ресурсов) количество клиентов. \
Игра заключается в отыскании выхода из лабиринта игроком за заданное число ходов. Сервер генерирует случайным образом несколько стен в «лабиринте», размер которого задается на сервере. 
Игрок стартует в левой нижней клетке, его задача — дойти до противоположного угла поля за заданное число ходов. Игрок может отправлять сообщения вида «вперёд», «направо», «налево», «назад»,
«сдаюсь». \
Сервер принимает сообщения игрока, проверяет их и на каждое сообщение присылает ответ («успешно», «там стена, осталось Y ходов», «вы проиграли», «вы выиграли»). После завершения игры сетевое соединение
завершается. Также предусмотрен режим тестирования без ограничений на число ходов игрока.
